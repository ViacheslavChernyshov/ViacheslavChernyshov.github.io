Telegram-бот на Python 3.11, aiogram 3, yt-dlp, ffmpeg и Whisper

Telegram-бот для скачивания видео и конвертации файлов

ToolboxBot принимает ссылки, файлы, голосовые и обычный текст, сам определяет тип контента и показывает только релевантные действия: скачать видео, вытащить аудио, сжать, конвертировать, распознать речь, собрать PDF, посчитать хэш или выгрузить результат в облако.

Функции, которые реально есть в репозитории

Ниже перечислены пользовательские действия, которые бот действительно показывает в интерфейсе на основе типа контента. Источник данных для этого лендинга — реальные `handlers`, `services`, `action_handlers`, локали и тесты проекта.

Скачать видео

Скачивайте видео с YouTube, TikTok, Instagram Reels, VK, X/Twitter и Facebook прямо в Telegram. Бот использует yt-dlp, спрашивает качество и для Pro открывает уровни 1440p, 2160p и 4320p.

  • YouTube
  • TikTok
  • Instagram
  • VK
  • X/Twitter
  • Facebook
  • MP4
  • 360p-4320p

Скачать аудио

Из медиассылки можно забрать только аудиодорожку и сразу выбрать целевой формат. В коде доступны MP3, OGG, M4A, FLAC и WAV.

  • MP3
  • OGG
  • M4A
  • FLAC
  • WAV
  • yt-dlp

Транскрибировать аудио и видео

Голосовые, аудиофайлы, видео и даже медиассылки переводятся в текст. Пайплайн сначала пробует Groq Whisper API, а при ошибке или лимите автоматически переключается на локальный faster-whisper.

  • Voice
  • Audio
  • Video
  • Groq
  • faster-whisper

Конвертировать видео

Видео, видеозаметки и GIF-анимации можно перекодировать через ffmpeg. Реально поддерживаются форматы MP4, MKV, AVI и WEBM.

  • MP4
  • MKV
  • AVI
  • WEBM
  • ffmpeg

Извлечь аудио из видео

Если нужен только звук, бот забирает аудиодорожку из видеофайла и сохраняет её в нужный формат. На выходе доступны MP3, OGG, M4A, FLAC и WAV.

  • MP3
  • OGG
  • M4A
  • FLAC
  • WAV

Сжать файл

Для фото, видео и аудио используется медиасжатие, а для документов, архивов, PDF, DOCX, TXT, CSV и бинарных файлов — ZIP-упаковка с тремя уровнями. Это одна из самых практичных функций Telegram-бота для файлов.

  • Pillow
  • ZIP high
  • ZIP med
  • ZIP low

Конвертировать аудио

Аудио и голосовые сообщения можно быстро перевести в другой формат. В проекте заданы MP3, OGG, M4A, FLAC и WAV без внешних GUI-программ.

  • OGG
  • M4A
  • FLAC

Конвертировать изображение

Фото, статичные стикеры и видеостикеры можно переводить между JPG, PNG, WEBP и BMP. Для сложных входов бот сначала пробует Pillow, а затем fallback через ffmpeg.

  • PNG
  • WEBP
  • BMP
  • Pillow

Изменить размер изображения

Для фото и документов-изображений есть быстрые сценарии -10%, -20%, -30% и -50%. На платных тарифах доступен ещё и ввод собственного максимального размера стороны.

Перевести в PDF

Фото конвертируются в PDF через Pillow, а TXT и DOCX — через LibreOffice. Это полезно, когда нужен простой Telegram-бот для PDF без установки редакторов на компьютер.

  • PNG
  • WEBP

Извлечь текст

Бот вытаскивает текст из PDF, TXT, CSV и DOCX. Для PDF используется pypdf, для DOCX — разбор XML внутри файла, а текстовые документы читаются напрямую.

  • PDF
  • TXT
  • CSV
  • DOCX
  • pypdf

Посчитать хэш файла

Для любого загруженного файла бот считает контрольные суммы MD5, SHA1, SHA256 и CRC32. Удобно для проверки целостности архива, документа, аудио или видео.

  • MD5
  • SHA1
  • SHA256
  • CRC32

Посчитать хэш текста

Обычный текст без файла тоже можно хэшировать прямо в Telegram. В ответ бот возвращает тот же набор: MD5, SHA1, SHA256 и CRC32.

  • Текст
  • MD5
  • SHA1
  • SHA256
  • CRC32

Создать QR-код

Любую ссылку или текст можно превратить в PNG с QR-кодом. Это работает для обычных URL, медиассылок, заметок, коротких инструкций и любых строк, которые пользователь отправил в чат.

  • URL
  • Текст
  • PNG
  • qrcode

Сократить ссылку

Если в сообщении есть URL, бот может пропустить его через TinyURL и вернуть короткий адрес. Это отдельное действие для распознанных ссылок и обычного текста с http.

  • TinyURL
  • URL
  • Текст

Пользовательский флоу

Как работает ToolboxBot

Пользовательский сценарий в коде очень прямой: бот получает контент, определяет тип, показывает встроенную клавиатуру действий, при необходимости запрашивает качество, формат, размер или уровень сжатия и запускает задачу в очереди.

  1. 1

    Запустите бота и откройте базовые команды

    Точка входа — команда /start. Для справки есть /help и /features, а состояние тарифа и остатка действий показывает /status.

    /start /help /features /status
  2. 2

    Отправьте ссылку, файл, голосовое или текст

    В коде поддерживаются сообщения типов text, photo, video, audio, voice, document, animation, sticker и video_note. Для документов бот дополнительно определяет подтипы: видео, аудио, изображение, PDF, текстовый документ или архив.

  3. 3

    Получите только те действия, которые подходят этому контенту

    Для YouTube-ссылки появятся кнопки скачивания видео, аудио, транскрибации, QR и сокращения URL. Для фото будут доступны сжатие, resize, конвертация, PDF, хэш и выгрузка, а для текста — QR, хэш текста и сокращение ссылок.

  4. 4

    Если нужно, выберите качество, формат или параметры

    Видео по ссылке получает экран выбора качества, а конвертация и извлечение аудио показывают список форматов. Для сжатия есть уровни high, med, low, а resize предлагает фиксированные проценты и премиум-режим с собственным размером.

  5. 5

    Задача идёт в очередь, а вы видите статус и результат

    В процессе бот меняет статусы на downloading, converting, compressing, transcribing, uploading или sending. На выходе приходит либо готовый файл в чат, либо внешняя ссылка, если результат слишком большой для Telegram.

Реальные команды, которые доступны пользователю

/status

Показывает текущий тариф, остаток действий и дату окончания подписки, если она активна.

/buy

Открывает покупку подписок Lite и Pro через Telegram Stars. Также работают алиасы /premium и /subscribe.

/donate

Позволяет поддержать проект фиксированной суммой или отправить свой размер доната через Stars.

/support

Открывает форму обратной связи с администраторами. В коде также есть команды /feedback и /contact.

Стек бота

Python 3.11, aiogram 3, yt-dlp, ffmpeg, Pillow, pypdf, qrcode, Groq, faster-whisper, SQLite, aiohttp, Loguru, Playwright и keyring. В Dockerfile также явно настроены Node.js и Chromium для Playwright-сценариев.

Политика конфиденциальности

По коду проекта входные и выходные файлы складываются во временные каталоги и очищаются после завершения задачи. Это снижает следы обработки на сервере и удерживает рабочее пространство бота в чистом состоянии.

Логи в ToolboxBot санитизируются: из них вырезаются токены, ключи, ID, usernames и секреты из URL. Дополнительно проект умеет читать секреты из keyring операционной системы, а не только из .env.
